FAQ

Questions fréquentes

Comment accéder à une formation après paiement ?▾
Après confirmation de votre paiement via WhatsApp, notre équipe vous envoie un code d'accès unique. Connectez-vous à votre compte Luckily Academy, allez sur la page de la formation, cliquez sur "J'ai déjà payé — Entrer mon code" et saisissez votre code. L'accès est immédiat et à vie.
Quels modes de paiement acceptez-vous ?▾
Nous acceptons MTN Mobile Money, Moov Money (Flooz), Wave, et les cartes bancaires Visa/Mastercard. Pour tout autre mode de paiement, contactez-nous directement via WhatsApp au +229 01 59 60 95 81.
Les formations sont-elles accessibles depuis mobile ?▾
Absolument ! La plateforme Luckily Academy est 100% responsive et optimisée pour tous les appareils : smartphones, tablettes, ordinateurs portables et fixes. Vous pouvez apprendre depuis n'importe quel endroit.
Comment fonctionne le code d'accès aux formations ?▾
Chaque formation a un code d'accès unique. Après votre paiement confirmé par notre équipe, vous recevez votre code personnel via WhatsApp. Ce code vous permet d'accéder uniquement à la formation payée. Il est strictement personnel et ne doit pas être partagé.
L'accès aux formations expire-t-il ?▾
Non ! Une fois inscrit(e), vous avez accès à votre formation à vie. Vous pouvez la suivre à votre rythme, la reprendre autant de fois que vous le souhaitez, et télécharger les supports PDF sans aucune limite de temps.
Y a-t-il un certificat à la fin des formations ?▾
Oui ! À la fin de chaque formation, vous recevez un certificat officiel Luckily Academy. Ce certificat est reconnu par les employeurs et peut être partagé sur LinkedIn ou présenté lors de vos candidatures professionnelles.
